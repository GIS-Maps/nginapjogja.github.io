FITUR PLATFORM

Modul Analisis Spasial

Pemrosesan data mentah menjadi informasi bernilai tspasial visual menggunakan algoritma GIS modern.

01 — GEOCODING

Pencarian & Lokasi

Cari hotel by nama, alamat, atau kecamatan. Geocoding real-time via Nominatim OSM API.

02 — ROUTING

Petunjuk Arah

Kalkulasi rute mengemudi dari lokasi Anda ke hotel pilihan menggunakan OSRM engine.

03 — BUFFER ANALYSIS

Kawasan Sekitar Hotel

Analisis buffer 500m untuk identifikasi restoran, ATM, rumah sakit, dan POI sekitar hotel.

04 — HEATMAP

Peta Kepadatan Hotel

Visualisasi densitas persebaran hotel berdasarkan tingkat hunian (capacity) per kawasan.

05 — GEODESIGN SCORE

Skor Keberlanjutan

Penilaian multi-variabel: konektivitas, beban layanan, kedekatan heritage, dan indeks vegetasi.

06 — EXPORT DATA

Unduh Dataset

Export data spasial dalam format JSON, CSV untuk analisis lanjut menggunakan QGIS atau ArcGIS.
